# 🦷 SmileBright Dental Clinic Landing Page

A modern and responsive landing page for a dental clinic, designed to provide patients with an engaging experience and showcase the clinic's services in a clean and professional way.

---

## 📌 Overview

This project is a static landing page built to represent a modern dental clinic website. It focuses on user experience, responsive design, smooth interactions, and clear presentation of the clinic's information.

The website is suitable for clinics, dentists, and healthcare businesses looking for a modern online presence.

---

## ✨ Features

- Modern and clean UI
- Fully responsive on all devices
- Smooth scrolling navigation
- Animated page elements
- Interactive testimonials slider
- FAQ accordion section
- Appointment booking form
- Contact section with clinic information
- Newsletter subscription
- Dark / Light mode
- SEO-friendly structure
- Accessibility best practices
